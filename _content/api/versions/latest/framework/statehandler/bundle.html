statehandler

History functionality and state saving.

Description

Handles state changes to keep track of history and provides functionality to save application state into a cookie/backend service.

Requests that the bundle handles

Request Use
StateHandler.SaveStateRequestRequests state to be saved
StateHandler.SetStateRequestRequests state to be set

Events that the bundle sends

Event Use
StateSavedEventUsed to notify that application state has been saved and any listing should refresh
